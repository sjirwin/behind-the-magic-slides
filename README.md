# Behind the Magic

This repo contains the content and tools used to create the slides for the conference talk [**Behind the Magic: Unlocking Python's Descriptor Protocol**](https://sjirwin.github.io/behind-the-magic-slides/).

The slide content is licensed under the terms and conditions contained in [LICENCE.md](LICENSE.md).

The code that is used during the live demo portion of the talk lives in the repo [`sjirwin/behind-the-magic`](https://github.com/sjirwin/behind-the-magic) and is covered by the license found there.

Version 5.2.1 of [reveal.js](https://revealjs.com/) was used to render the slides and this repo contains a copy of that code. The LICENSE and README files for version 5.2.1 of the project are preserved here as [LICENSE-reveal.js.md](LICENSE-reveal.js.md) and [README-reveal.js.md](README-reveal.js.md). The current version of the `reveal.js` LICENSE and README files can be found in the [`hakimel/reveal.js`](https://github.com/hakimel/reveal.js) repo.
