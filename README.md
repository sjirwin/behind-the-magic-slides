# Behind the Magic

This repo contains the content and tools used to create slides for the conference talk [**Behind the Magic: Unlocking Python's Descriptor Protocol**](https://sjirwin.github.io/behind-the-magic-slides/).

The slide content is licensed under the terms and conditions contained in [LICENCE.md](LICENSE.md).
